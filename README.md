---

## 🛠️ Project Setup

### Prerequisites

- [Node.js](https://nodejs.org/) (v16+ recommended)
- [npm](https://www.npmjs.com/) (comes with Node)

### Install dependencies

```bash
npm install
```

### Run Locally

```bash
npm start
```

### Deploy on the server
```bash
npm deploy
```
